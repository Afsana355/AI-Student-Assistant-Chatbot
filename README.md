# AI-Student-Assistant-Chatbot

An AI Student Assistant Chatbot developed using Python, Tkinter, and NLTK for answering student-related queries.

## 🚀 Features
* **Interactive GUI:** Built with Tkinter for a user-friendly interface.
* **Natural Language Processing:** Uses NLTK for tokenization, stemming, and pattern recognition.
* **Customizable Responses:** Uses an `intents.json` file to easily update tags, patterns, and responses.
* **Student Support:** Answers common queries regarding admissions, courses, and schedules.
